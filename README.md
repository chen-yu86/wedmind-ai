# WedMind AI 婚禮規劃系統 WedMind AI Wedding Planning System
WedMind AI 是一個婚禮規劃系統，協助新人整理婚禮流程、預算與準備清單，讓婚禮籌備可以更有效率地管理。

本專案提供婚禮流程時間軸、預算分析與婚禮準備清單功能，並透過資料視覺化圖表讓使用者更容易了解婚禮花費比例。

## 功能 Features

### 婚禮流程時間軸
整理訂婚迎娶與婚宴流程，並以時間軸方式呈現

### 婚禮預算分析
透過 Chart.js 建立婚禮預算圓餅圖，清楚顯示各項花費比例

### 預算互動調整
使用滑桿調整餐廳預算，圖表會即時更新

### 婚禮準備清單
提供婚禮物品 checklist 並顯示完成進度

### AI 預算建議
使用者輸入婚禮預算後，系統會提供建議花費比例
