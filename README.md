# WedMind AI 婚禮規劃系統 WedMind AI Wedding Planning System
WedMind AI 是一個婚禮規劃輔助系統
提供婚禮流程管理、預算分析與婚禮準備清單功能
此專案為個人作品集專案，用於展示前後端整合能力

## 功能
- 婚禮流程時間軸
- 婚禮預算分析圖表
- 婚禮準備清單
- 任務進度追蹤

## 技術
Frontend
- HTML
- CSS
- JavaScript
- Chart.js

Backend (可延伸)
- FastAPI
- SQLite
